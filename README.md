## 파이썬 
1. 파이썬 라이브러리 설치 시 
- window: pip 명령어 사용
예) pip install requests
- MacOS: pip3 명령어 사용 (Mac에서 pip install xxx를 실행하면 에러가 발생한다) 
예) pip3 install requests
